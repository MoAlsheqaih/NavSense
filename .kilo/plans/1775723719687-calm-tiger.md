# Fix Floor Plan Container Overflow Issue

## Problem
The floor plan is not properly contained within its container. Parts of the map are rendering outside the visible container area.

## Root Cause
The current scaling and offset calculation in `FloorGridPainter` and `SimulationMapWidget` doesn't properly constrain the floor plan to stay within the container bounds. When zooming or panning, or even at default scale, the floor plan can overflow the container boundaries.

## Solution Plan

### 1. Fix FloorGridPainter Scaling
**File:** `navsense/lib/presentation/pages/simulation/widgets/floor_grid_painter.dart`

**Changes needed:**
- Add padding around the floor plan within the container
- Ensure scale calculation leaves margin for boundaries
- Fix offset calculation to prevent overflow at all scale levels
- Add minimum padding constant (e.g., 20px on each side)

**Current problematic code:**
```dart
final scaleX = size.width / floorWidth;
final scaleY = size.height / floorHeight;
final scale = math.min(scaleX, scaleY);

final offsetX = (size.width - floorWidth * scale) / 2;
final offsetY = (size.height - floorHeight * scale) / 2;
```

**Should be changed to:**
```dart
// Add padding to prevent edge overflow
const double padding = 20.0;
final availableWidth = size.width - (padding * 2);
final availableHeight = size.height - (padding * 2);

final scaleX = availableWidth / floorWidth;
final scaleY = availableHeight / floorHeight;
final scale = math.min(scaleX, scaleY);

// Use padding as minimum offset
final offsetX = padding + (availableWidth - floorWidth * scale) / 2;
final offsetY = padding + (availableHeight - floorHeight * scale) / 2;
```

### 2. Fix SimulationMapWidget Zoom/Pan Constraints
**File:** `navsense/lib/presentation/pages/simulation/widgets/simulation_map_widget.dart`

**Changes needed:**
- Update `_constrainPan()` method to use proper bounds
- Add minimum scale constraint to prevent over-zoom
- Ensure pan limits account for the padding added in FloorGridPainter
- Fix container size calculation for proper constraints

### 3. Verify Container Layout
**File:** `navsense/lib/presentation/pages/simulation/simulation_map_page.dart`

**Changes needed:**

- Verify the Expanded widget properly fills available space
- Check that the map widget receives correct size from parent
